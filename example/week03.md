# Week03. [clang/IR Pass] 소스 코드에 존재하는 함수 호출 개수 출력하기

## **Requirement**
1. **[기본]** clang 컴파일러 프론트엔드에 옵션(ex. -fprofile-collect-func-info) 추가하기
2. **[기본]** 사용자 정의 LLVM IR Pass 에서는 소스코드에 존재하는 함수 호출 명령 개수 출력하기
    * **[가정]** : printf 함수는 소스코드에서 이미 사용하였음
<br><br>

## **Example**
### *- 예제 소스코드*
``` C
// example.c
#include <stdio.h>

void func()
{
}
int main()
{
        printf("Hello\n");
        func();
        return 0;
}
```
<br>

### *- 수행 방법*
``` bash
# 기본, clang 컴파일러에서 옵션을 사용하여 example.c 소스코드를 Human Readable LLVM IR 코드로 변환하기
$ clang -fprofile-collect-func-info example.c -emit-llvm -S -c -o example.mod.ll

# 기본, 수정된 IR 파일을 컴파일하여 실행해보기
$ clang -o example example.mod.ll
$ ./example
...
PROFILE_COLLECT_FUNC_INFO-----
func: 1
printf: 1
```
<br>
